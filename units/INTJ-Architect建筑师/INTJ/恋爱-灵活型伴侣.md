# 恋爱 — 与灵活型伴侣

与计划型不同，灵活型喜欢保持选择的开放性，总是寻找替代方案和机会。除非双方都花时间去探索和理解彼此的差异，否则计划型与灵活型之间的关系将充满挑战。

这种关系中主要的不满源于你们在制定和执行计划上的对立观点。作为计划型，你会制定计划并竭尽全力坚持执行。而作为灵活型，你的伴侣则更倾向于灵活自发地行动——甚至可能在完成最有趣的任务后就放弃整个项目。

然而，计划型与灵活型的关系也存在潜在优势。计划型固执带来的紧张感，可以被随和的灵活型化解。同样，你的进取心和条理性，可以平衡伴侣优柔寡断的倾向，或其对那些不有趣、不愉悦的任务视而不见的倾向。只要双方避免互相指责并愿意妥协，你们之间的差异就能转化为巨大的优势，因为你们各自都能弥补对方的劣势。

为了实现这种和谐，你们应努力相互妥协。作为计划型一方，你应偶尔尝试新事物或表现得更随性一些，即使这意味着改变你的习惯或长期持有的观点。同样，作为灵活型一方，你的伴侣也应克制寻求新奇事物以及逃避必要计划或家务的冲动。你们的稳定与伴侣的随性之间需要保持平衡，而你们双方都必须共同决定这种平衡在你们的关系中应处于何种位置。

## 给计划型的建议

1. **沟通。** 伴侣的优柔寡断或随性态度常会令你感到沮丧。切勿恶语相向，试着理解伴侣的动机，并思考如何互相支持。请记住，伴侣可能不像你那样热衷于计划和日程安排。

2. **保持开放心态。** 作为计划型，你可能难以接受他人的观点。当伴侣与你意见相左时，尽量不要固执己见或争辩不休。

3. **勇于尝试。** 尽管你习惯于规划和准备，但请记住，有些情况需要尝试或做些新尝试。学会适时放手，偶尔随性而为。

4. **不要透支自己。** 你渴望把事情做完的欲望可能会给自己和伴侣都带来压力。试着放松，并对某些自己设定的截止日期采取灵活的态度。

## 给灵活型的建议

1. **沟通。** 伴侣的固执和对计划的执着往往会让你感到沮丧。切勿诉诸人身攻击。相反，试着去理解伴侣，并思考如何互相支持。请记住，你的伴侣习惯于将计划和日程安排置于娱乐与休息之上，对方可能难以退后一步放松身心。

2. **专注。** 作为灵活型，你可能喜欢尝试新事物和改变方向。学会完成任务并履行责任。

3. **关注环境。** 你对杂乱无序的容忍度可能远高于你的计划型伴侣。不要忽视家务职责。

4. **不要挑三拣四。** 灵活型往往抗拒例行公事和琐碎任务。你可能会想把注意力转向更刺激的事情，但你应该有意识地努力承担自己应尽的家庭责任。
